Data should be treated as a function in its own right, serving the other functions in the business.

Financial and operational systems, web application databases, event and log streams, and third party SaaS services all generate data, and the tools and methods by which all this data is tracked, gathered, analyzed, and surfaced is evolving at a frenetic pace.

I help organizations focus on the insights they need most urgently and build a well-tuned data operation — even if it is one person to start — to ensure data capture, transformation, storage, and reporting respond as quickly as the business.
